    \section{kernel/dispath.c文件}
    
    dispatch函数主要逻辑：
    \begin{enumerate}
        \item 检查参数dcb是不是当前dcb,如果不是切换到参数dcb，包括上下文
        \item 更新disp的系统时间为systiem\_now + kernel\_off
        \item 根据disp的状态：disable执行resume，enable执行excute(调度线程)。
    \end{enumerate}
